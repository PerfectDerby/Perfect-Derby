# Baseball Game APK - Download Instructions

## 🎮 How to Download the APK

Every time you push code to GitHub, a new APK is automatically built!

### Method 1: Download from GitHub Actions (Latest Build)

1. Go to your repository: https://github.com/AliCruz1/baseball-game
2. Click on the **"Actions"** tab at the top
3. Click on the most recent **"Build Android APK"** workflow run
4. Scroll down to **"Artifacts"** section
5. Click **"baseball-game-apk"** to download the APK
6. Unzip the downloaded file to get `app-debug.apk`

### Method 2: Download from Releases (Tagged Versions)

1. Go to: https://github.com/AliCruz1/baseball-game/releases
2. Click on the latest release
3. Download `app-debug.apk` from the Assets section

## 📱 Installing on Your Phone

1. **Transfer the APK** to your Android phone
   - Email it to yourself
   - Use a cloud service (Google Drive, Dropbox)
   - Connect via USB and copy directly

2. **Enable Unknown Sources**
   - Go to Settings → Security → Unknown Sources (enable)
   - Or Settings → Apps → Special Access → Install unknown apps

3. **Install the APK**
   - Open the APK file on your phone
   - Tap "Install"
   - Open the app when installation completes
